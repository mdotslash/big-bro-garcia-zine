# Michael Garcia is Watching You! 

Written by Lis from the Community Privacy Residency and Mayowa Tomori and hosted using [Electric Zine Maker](https://alienmelon.itch.io/electric-zine-maker) zines on itch.io.

## How to Print / Edit the Zine 

- Print the PDF [here](pages/CommunityPrivacyZine-SantaFe_MichaelGarcia.pdf) and fold, distribute
- Edit / remix the zine in Powerpoint or Google Docs using [this template](https://docs.google.com/presentation/d/1ssUOuqe8cbBt9-9jNAOsoEz0AaBfHXGpgVhfe4j7QPg/edit?usp=sharing)
